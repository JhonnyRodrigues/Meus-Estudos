Tags em HTML

Combinando tag lista de definição com tag de abreviação

Em HTML5, podemos inserir os elementos da lista a seguir:
(passe o mouse sobre os termos para descobrir qual tag deverá utilizar em cada caso)

  • Parágrafos
    Use a tag <p>
  • Títulos
    Use as tags de <h1> à <h6>
  • Imagens
    Use a tag <img>
  • Abreviações
    Use a tag <abbr>
  • Códigos-fonte
    Combine as tags <pre> com <code>
  • Listas numeradas
    Use a tag <ol>
  • Lista com demarcadores
    Use a tag <ul>
