Principais Formatações

Negrito / Destaque

Nesta frase, temos um termo em negrito usando a tag <b> (não semântica).

Já nesta frase, temos um termo em destaque usando a tag <strong> (semântica).

Itálico / Ênfase

Nesta frase, temos um termo em itálico usando a tag <i> (não semântica).

Já nesta frase, temos um termo em ênfase usando a tag <em> (semântica).

Texto marcado

Podemos criar também um termo marcado usando a tag <mark>.

E nesse outro parágrafo, temos outro termo marcado no final.

Texto grande e pequeno

Estamos criando um texto grande com uma tag obsoleta e um texto pequeno com a tag <small>

Texto deletado

Usando a tag <del> podemos marcar um texto como excluído para indicar que ele deve ser lido, mas não considerado.

Texto inserido

Usando a tag <ins> podemos marcar um texto como inserido para dar ênfase e indicar que ele foi adicionado depois.

Existe também o sublinhado com a tag <u> (não semântica).

Texto sobrescrito

Para inserir caracteres elevados como: x20+3, usamos a tag <sup>

Textosubscrito

Para inserir caracteres rebaixados como: H2O, usamos a tag <sub>
